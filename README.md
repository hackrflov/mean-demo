# Prerequestion
- download npm
- download mongod

# Get Started

create an repository
```
  git clone https://github.com/hackrflov/mean-demo.git
```
install node modules
```
  npm install
```
install bower components 
```
  bower install
```
install mongod
```
  mongod --dbpath ./data
```
build project
```
  gulp
```
run program
```
  npm start
```
